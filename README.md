## How to run the code

To run models in the `models` subdirectory, you may need to add the top-level
`/models` folder to the Python path with the command:

```
export PYTHONPATH="$PYTHONPATH:/path/to/models"
```
Then,
```
cd models/official/resnet
./run.sh
```
